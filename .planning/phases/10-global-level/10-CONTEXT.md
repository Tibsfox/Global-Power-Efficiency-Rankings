<decisions>
## Implementation Decisions

### Prompt style/framing
- **Tone varies by pathway type:** Visionary/strategic for think tanks and academia; diplomatic/collaborative for UN/multilateral roles
- **Framework references:** Keep broad ("international climate commitments") rather than name-dropping specific treaties (Paris Agreement, SDGs) — shows domain awareness without sounding textbook
- **Team contribution framing:**
  - Secretariat context for UN roles ("contributing to UNFCCC technical paper")
  - Cross-border collaboration context for diplomatic/advisory roles ("G20 working group member")

### Claude's Discretion: Prompt patterns
- Claude decides whether to reuse National-level patterns (Program Navigation, Cross-Sector Analysis, etc.) with international framing or create Global-specific vocabulary — choose what fits each pathway

### International org coverage
- **Match org to pathway:** Policy roles get UN system, finance roles get development banks, technical roles get IEA/IRENA
- **Private sector included but not emphasized:** Global consulting (McKinsey sustainability, Big 4 climate) mentioned as career destinations, but prompts/projects focus on public institutions
- **Geographic balance per pathway:** Some naturally hub-centric (UN system = Geneva/NYC), others more distributed (regional development banks)
- **Entry programs in projects, not prompts:** JPO, YPP, consultant rosters appear as project ideas ("research and apply to 2 JPO positions"), not in learning prompts

### Skill emphasis
- **Cross-cultural vs negotiation:** Technical roles emphasize cross-cultural communication; policy roles emphasize multilateral negotiation — pathway-dependent
- **Language context:** Mention multi-language environments where specifically relevant (UN, EU) vs English-only orgs (IEA)
- **Soft power skills:** Claude decides emphasis by pathway — advisory/diplomatic roles get relationship-building focus, technical roles stay domain-focused
- **Network building:** Only at Proficient level — Beginner/Intermediate focus on deliverable skills
- **Development context:** Frame as "development effectiveness" and outcomes/impact, not geopolitical dynamics
- **International experience gap:** Address all three approaches:
  - Acknowledge the barrier exists
  - Focus on what's actionable from current position
  - Beginner projects include ways to gain international exposure without relocating
- **Remote work:** Mention where accurate per org type — some roles genuinely remote-possible, others require in-country presence
- **Career timelines:** Acknowledge implicitly through project complexity progression, don't explicitly state "5-10 years to senior"

### Resource priorities
- **Mixed resource types:** Publications for domain knowledge (IEA WEO, IRENA reports), courses for structured learning (UN Climate Learning, World Bank OLC)
- **Paid resources included:** Where valuable, include with cost note — some certifications have real value
- **Data portals:** Both as general resources (where data fluency needed) and in project contexts
- **Non-degree focus:** This report is about self-directed learning, not degree program recommendations

</decisions>

## Key Terminology Defined

| Term | Definition |
|------|------------|
| IOU | Investor-owned utility, PUC-regulated, serves ~72% of U.S. customers |
| DSM | Demand-side management - programs that reduce customer energy use |
| DR | Demand response - reducing electricity use during peak periods |
| TRC | Total Resource Cost test - cost-effectiveness methodology |
| Propensity model | ML model predicting customer participation likelihood |
| LMP | Locational marginal pricing - electricity prices by location/time |
| IRP | Integrated Resource Plan - long-term utility planning document |
| Intervenor | Non-utility party participating in regulatory proceedings |

<task type="auto">
  <name>Task 3: Formatting consistency audit across all 25 cards</name>
  <files>index.html</files>
  <action>
Audit formatting consistency across all 25 AI cards (5 levels x 5 cards each).

**Check for consistency in:**

1. **Structure:**
   - All cards use `.strategy-section` container
   - All cards have: strategy-header, strategy-content, strategy-stats, action-list, ai-sidebar, case-studies, project-ideas, career-transitions
   - Proper nesting and closing tags

2. **Case Studies:**
   - Each card has exactly 3 case studies
   - Case study format: `<strong>Organization/Program (Year)</strong>` followed by description
   - Measurable outcomes included (percentages, dollar amounts, timeframes)
   - Footnote citation format: `<sup><a href="#prefix-ref-N" role="doc-noteref">[N]</a></sup>`
   - Case study borders use level-appropriate color variable

3. **Project Ideas:**
   - DIY projects include: Data sources (with free status), Skills needed, Difficulty level, Cost
   - Commercial concepts include: Target market, Revenue model, Value proposition
   - Clear separation between DIY and Commercial sections

4. **Career Transitions:**
   - Each card has exactly 5 career paths
   - Format: `<strong>From Role → To Role</strong>`
   - Background, Skills to add, Resources (with links), Timeline included
   - Resource links open in new tabs (`target="_blank"`)

5. **Color Scheme:**
   - Individual: green (`var(--accent-green)`)
   - Community: blue (`var(--accent-blue)`)
   - State/Regional: cyan (`var(--accent-cyan)`)
   - National: amber (`var(--accent-amber)`)
   - Global: cyan (`var(--accent-cyan)`)

6. **Terminology:**
   - Acronyms explained inline on first use per section
   - Consistent capitalization of organizations (IEA, IRENA, UNFCCC, etc.)

**Fix any inconsistencies found:**
- Missing elements
- Incorrect formatting
- Color mismatches
- Missing footnote links
- Broken HTML structure

<task type="auto">
  <name>Task 2: Rewrite problematic prompts for variety</name>
  <files>index.html</files>
  <action>
For each prompt identified in sameness-audit.txt as needing rewrite:

1. Vary the prompt structure using these patterns:
   - **Persona-Based**: "Act as a [role] helping me transition from [current] to [target]..."
   - **Scenario-Based**: "I'm preparing for an interview at [org type]. Help me..."
   - **Project-Based**: "I want to build a portfolio project demonstrating [skill]..."
   - **Socratic**: "What skills would I need to develop to transition from [current] to [target]?"
   - **Challenge-Based**: "Given my background in [current field], what unique value could I bring to [target role]?"

2. Rewrite rules:
   - No two adjacent pathways (within same card) should use same pattern
   - Cross-level pathways (same card position across levels) should vary
   - Maintain functional equivalence - prompt should still generate useful career transition guidance
   - Keep pathway-specific context (role names, organizations, skills) but vary structure

3. For each rewrite:
   a. Read the original prompt in context (understand what it's trying to accomplish)
   b. Choose a different pattern from adjacent pathways
   c. Rewrite to maintain information content but vary delivery
   d. Verify rewrite still fits the HTML structure (within prompt-example div)

4. After all rewrites, re-audit to confirm variety:
```bash
grep -A 3 'AI Learning Prompt Example' index.html | \
    grep -oP '<p style="font-style: italic.*?</p>' | \
    sed 's/<[^>]*>//g' | \
    cut -c1-50 | sort | uniq -c | sort -rn | head -20
```
(Top patterns should each appear ≤3 times)

5. Commit the changes:
```bash
git add index.html
git commit -m "fix(qa): vary prompt structure across pathways

Rewrote [N] prompts to eliminate identical/templated content.
Patterns now varied: Persona-Based, Scenario-Based, Project-Based, Socratic, Challenge-Based.
Maintained functional equivalence while improving content variety.

Co-Authored-By: Claude Opus 4.5 <noreply@anthropic.com>"
```
